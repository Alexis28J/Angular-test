* Lezione di mercoledì 28 gennaio 2026

1. Abbiamo eseguito "ng new nome-applicazione --directory ./" sul terminal
Questo commando serve a creare un nuovo progetto Angular chiamato nome-applicazione nella cartella corrente (indicata da ./), 
invece che in una nuova sottocartella omonima, generando tutti i file e le configurazioni necessarie per iniziare a sviluppare l'applicazione Angular immediatamente nella posizione specificata, 
senza creare una cartella aggiuntiva. 

2. Invio sull'opzione SCSS (SCSS, a differenza di CSS, permette di utilizzare funzionalità avanzate come variabili, annidamento e mixin per scrivere fogli di stile più modulari e manutenibili.)

3. Scrivere N (No) e premere invio

4. Invio sull'opzione None
